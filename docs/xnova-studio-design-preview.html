这版和旧预览的差异

  • 保留了渐变、玻璃感、阴影和大圆角，但整体收成更接近 Codex App 的安静工作台气质。
  • 首页建议动作从 feature cards 收成轻量列表项，更符合“立即开工”的主叙事。
  • Context Bar 改成更紧凑的 HUD strip，不再像六张并排卡片。
  • 布局严格贴合当前 `studio/` 里的主链路：项目树、Scratchpad、Mode、Context、SubAgent、Tools/Settings 状态页。
